What I learned in this course of Coursera for Git and GitHub

1. git stash

  • Meaning: Temporarily saves your uncommitted changes so you can work on something else.
  • Commands:
    • git stash
    • git stash pop
    • git stash list
    • git stash apply

2. git rebase

  • Meaning: Reapplies commits on top of another base branch.
  • Commands:
    • git rebase <branch>
    • git rebase -i <branch>

3. git log

  • Meaning: Shows commit history.
  • Commands:
    • git log
    • git log --oneline
    • git log --graph

5. GitHub Features

  • Wiki: Documentation space for projects
  • Projects: Task tracking (like Trello)
  • Insights: Repository analytics
  • Actions: Automate workflows (CI/CD)
  • Issues: Track bugs and tasks

6. Useful Terminal Commands

  • clear — Clear terminal
  • pwd — Show current directory
  • ls — List files
  • cd <folder> — Change directory
  • history — Show previous commands
